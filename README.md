# libvirt-java

The `libvirt` JNA bindings. Forked from `http://libvirt.org/git/?p=libvirt-java.git`.

### Build

After running the `package` command the JAR will be located in the `target` directory.

```
mvn clean package
```

## Running the tests

```
mvn clean test
```
